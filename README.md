Calculadora de Evento

Projeto simples em Java para calcular a quantidade de itens necessários para um evento.

📌 Funcionalidades

- Cálculo de copos
- Cálculo de salgados
- Cálculo de doces
- Cálculo de bolo
- Cálculo de bebidas

🛠️ Tecnologias

- Java

▶️ Como executar

1. Compile:

javac src/Main.java

2. Execute:

java -cp src Main

📚 Objetivo

Projeto desenvolvido para praticar lógica de programação e Java.
